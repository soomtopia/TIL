## Boolean 자료형

boolean 은 참/거짓을 나타내는 자료형이에요. 값이 True 또는 False로만 이루어지죠.

- 참은 True

- 거짓은 False 입니다

- 수학에서도 대소 비교가 있듯,  파이썬에서도 __비교__ 연산이 존재해요. 이연산을 한 값은 __boolean__ 값으로 나옵니다

  - < , <=, >, >=, ==(같다), !=(같지않다) 이 있어요 

  - 값이 아닌 "주소"를 비교할 때가 있어요. 주소란 값이 들어있는 주소를 말해요. `id()` 를 사용하면 주소를 알 수 있어요

    ```python
    >>> a = []
    >>> b = []
    >>> id(a)
    4541295688
    >>> id(b)
    4541296456
    >>> a == b
    True
    >>> a is b
    False
    ```

  - 논리 연산자도 있어요

    - and 는 모든 값이 참일 경우만 True
    - or 는 하나의 값만 참이여도 True 
    - not 은 값의 반대를 출력해줘요

  - 숫자가 0 일 경우 False 라고 처리해요. `bool() `메서드는 괄호 안의 값의 참 거짓을 판단해줘요.

    ```python
    >>> bool(0)
    False
    >>> bool(2)
    True
    ```

  - 문자의 경우 빈 문자열은 False 라고 처리해요 

    ```python
    >>> bool('')
    False
    >>> bool('zz')
    True
    ```
